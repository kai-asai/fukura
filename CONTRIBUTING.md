# Contributing to fukura

IssueやPull Requestを歓迎します。

## Issue

- 不具合はOS、OSバージョン、fukuraのバージョン、再現手順を記載してください。
- 機密情報、個人の辞書、署名情報を添付しないでください。
- セキュリティ問題は公開Issueにせず、`SECURITY.md` に従ってください。

## Pull Request

1. Issueまたは変更目的を明確にします。
2. 変更範囲を小さく保ちます。
3. 関連する自動テストと手動テストを追加・更新します。
4. `node --test tests/*.test.mjs` と対象OSのビルドを実行します。
5. ユーザー向けの変更はREADMEやCHANGELOGにも反映します。

Pull Requestを送ることで、提出したコードをこのリポジトリのApache License 2.0の下で配布することに同意したものとします。

## 表記とプライバシー

- 公開文言では他社製品との比較・模倣を示す表現を避けます。
- ログへ入力内容、辞書本文、クリップボード内容を記録しません。
- `fukura`の名称とブランド資産の扱いは `TRADEMARKS.md` に従います。
