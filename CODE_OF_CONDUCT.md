# Code of Conduct

fukuraコミュニティでは、経験、国籍、性別、障害、外見、宗教、その他の属性にかかわらず、すべての参加者に敬意をもって接します。

## 期待する行動

- 建設的で具体的なフィードバックを行う
- 異なる経験や意見を尊重する
- 誤りを認め、改善に協力する
- 利用者の安全とプライバシーを優先する

## 認められない行動

- 差別、嫌がらせ、脅迫、個人攻撃
- 個人情報や非公開情報の無断公開
- 議論を意図的に妨害する行為
- その他、協働の場として不適切な行為

違反を見つけた場合は公開の場で応酬せず、GitHubのReport機能を利用してください。メンテナーは必要に応じて投稿の編集・削除、参加制限などの対応を行います。
